Format strict : Chaque question doit être un objet JavaScript avec ces champs :

category : la catégorie exacte que tu donnes

question : texte exact de ta question

answers : exactement 4 propositions (mauvaises ou bonnes)

correct : tableau des indices des bonnes réponses (0 à 3), même s’il y en a une seule ou plusieurs

explanation : explication neutre, factuelle et vérifiable

Règles de contenu :

Se baser uniquement sur le droit français officiel

Les réponses incorrectes doivent être plausibles mais fausses

Si plusieurs réponses sont correctes, proposes 2 ou 3 réponses et toutes doivent être listées dans correct

Les réponses correctes doivent être placées de façon aléatoire dans le tableau answers

Syntaxe JavaScript valide :

Pas de markdown dans les strings

Les guillemets internes dans une string doivent être échappés avec \"

Toujours 4 propositions par question

Autres points :

Ne jamais commencer par const questions = ni mettre de crochets globaux

Ne pas ajouter de texte ou commentaire supplémentaire

Mélanger l’ordre des bonnes réponses pour que ce ne soit pas toujours l’index 0

En résumé : fichier JS propre, 4 réponses par question, bonnes réponses aléatoires, explications correctes et vérifiables, syntaxe valide.
